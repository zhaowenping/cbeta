字體安裝

Linux

  1. 下載並存取hanazono-20170904.zip此一文檔。
  2. 打開hanazono-20170904.zip。
  3. 按兩下HanaMinB.ttf此一文檔。
  4. 按「安裝字體」。

Mac OS X

  1. 下載並存取hanazono-20170904.zip此一文檔。
  2. 打開hanazono-20170904.zip。
  3. 按兩下HanaMinA.ttf此一文檔。
  4. 按「安裝字體」。
  5. 按兩下HanaMinB.ttf此一文檔。
  6. 按「安裝字體」。

Windows

  1. 下載並存取hanazono-20170904.zip此一文檔。
  2. 打開hanazono-20170904.zip並把字體HanaMinA.ttf及HanaMinB.ttf存到電腦的某一個資料夾。
  3. 按一下 [開始]，再按一下 [執行]。
  4. 輸入 %windir%\fonts，然後按一下 [確定]。
  5. 按一下 [檔案] 功能表上的 [安裝新字型]。
  6. 按兩下包含字體的資料夾。
  7. 選取HanaMinA 及 HanaMinB兩個字體。
  8. 按一下以選取 [將字型複製到字型資料夾] 核取方塊。
  9. 按一下 [確定]。

IOS

  1. 在 iOS 7 及以上版本中，安裝字體需要先從 App Store 下載 AnyFont(或者RightFont) 或者 WorkFlow。把需要的「花園精簡」字體(從花園中抽取，只有4.7M適合蘋果移動設備)和悉檀字體通過分享打開，點擊字體後再點開「 設置」 ，安裝描述文件。
  2. 具體的方法可以參考 這篇文章 。iOS 6 及以前版本不支持安裝字體。

Android

  1. 直到 Android 7.1，Google 還不允許用戶安裝字體。某些 ROM 允許用戶替換系統默認字體，詳情請咨詢相關客服。
